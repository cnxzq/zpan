# Ark Pan - 项目介绍

Ark Pan 是一个轻量级带基础认证的静态文件网盘系统，基于 Node.js + Express 构建，支持文件上传、目录浏览和内网穿透访问。

## 什么是 Ark Pan

Ark Pan 设计用于快速搭建一个带密码保护的个人文件共享服务。它特别适合：

- 个人文件共享和备份
- 团队内部文件分享
- 通过内网穿透将本地文件暴露到公网访问
- 需要简单认证保护的静态文件服务

## 核心特性

| 特性 | 说明 |
|------|------|
| 📁 **静态文件托管** | 直接托管本地目录中的文件，支持浏览器访问下载 |
| 📂 **目录浏览** | 自动生成目录列表页面，支持图标展示 |
| 🔐 **基础认证** | 内置 Basic Auth 账号密码保护 |
| 📤 **文件上传** | 支持通过网页界面上传文件，最大支持 10GB |
| ⚙️ **灵活配置** | 支持命令行参数、环境变量、.env 文件多种配置方式 |
| 📡 **内网穿透友好** | 完美配合 frp 等内网穿透工具使用 |
| 🚀 **轻量快速** | 基于 Express，启动迅速，资源占用低 |

## 技术栈

- **运行时**: Node.js (>= 16)
- **框架**: Express 4.x
- **语言**: TypeScript
- **打包工具**: Rolldown
- **核心中间件**:
  - `express-basic-auth` - 基础认证
  - `multer` - 文件上传处理
  - `serve-index` - 目录浏览

## 使用场景

### 个人网盘
在本地或 VPS 上启动，将你的文件目录变成可访问的网盘，支持上传下载。

### 内网穿透
配合 frp 将本地服务暴露到公网，随时随地访问你家里电脑上的文件。

### 临时文件分享
快速启动一个带密码保护的文件分享服务，分享给团队成员使用。
